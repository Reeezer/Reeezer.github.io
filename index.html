Projects

Legal Document QA with Retrieval-Augmented Generation @ EPFL

Built a Retrieval-Augmented Generation (RAG) system to answer queries over EPFL legal documents by integrating semantic search and Large Language Models. The pipeline uses text chunking, FAISS vector indexing, and real-time retrieval through LangChain. Designed custom prompt templates and interactive agents to generate grounded, context-aware answers, enhancing transparency and usability in legal document navigation.

Machine Learning Retrieval-Augmented Generation Natural Language Processing Semantic Search

Based on LangChain, FAISS & OpenAI APIs

Personnalized Spinal Cord Reconstruction @ .NeuroRestore

Conduct research toward the development of a novel foundation model for spinal MRI, enabling accurate 3D reconstruction of patient-specific spinal cord anatomy to support surgical planning. The approach employs a self-supervised 3D Masked Autoencoder architecture based on Vision Transformers (ViT) to learn rich anatomical representations from unlabeled data. These representations are subsequently fine-tuned for downstream tasks such as spinal root segmentation and correction of MRI bending artifacts.

Machine Learning Neuroimaging Foundation Model Research

Based on ViT Masked Autoencoders

AI Tutor for Scientific Courses @ EPFL

Developed an AI tutoring system to assist students in understanding scientific concepts through interactive explanations and personalized support. The language model was fine-tuned using Parameter-Efficient Fine-Tuning (PEFT) methods alongside Supervised Fine-Tuning (SFT), Direct Preference Optimization (DPO), and Reinforcement Learning from Human Feedback (RLHF) to align outputs with student needs and academic standards. The system delivered accurate, context-aware answers and step-by-step solutions across a wide range of STEM problems.

Machine Learning Large Language Model Reinforcement Learning Model Alignment Parameter-Efficient Fine-Tuning

Based on OpenAI GPT2

Decoding Brain Signals for Movement Recovery @ .NeuroRestore

Developed deep learning models to decode arm movement intentions from electrocorticography (ECoG) signals, contributing to neurotechnological interventions for spinal cord injury (SCI) rehabilitation. Investigated architectures including EEGNet and CEBRA to improve the spatiotemporal representation of motor-related neural dynamics.

Machine Learning Time Series Fine-Tuning

Based on EEGNet & CEBRA
